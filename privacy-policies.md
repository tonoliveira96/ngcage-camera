# Política de Privacidade

Última atualização: 17 de Dezembro de 2025

Esta Política de Privacidade descreve como o aplicativo **Ncg Camera**, desenvolvido por **TonOliveiraDev**, coleta, usa e protege as informações dos usuários.

Ao utilizar este aplicativo, você concorda com os termos descritos nesta política.

---

## 1. Informações que coletamos

O **Ncg Camera** **não coleta, armazena ou compartilha dados pessoais identificáveis** dos usuários.

O aplicativo pode acessar os seguintes recursos do dispositivo **apenas para funcionamento local**:

- **Câmera**: utilizada exclusivamente para captura de imagens dentro do próprio aplicativo.
- **Armazenamento local**: utilizado para salvar imagens geradas pelo usuário no dispositivo.
- **Processamento de imagem**: realizado **inteiramente no dispositivo**, sem envio de dados para servidores externos.

Nenhuma imagem, vídeo ou informação é enviada para a internet ou armazenada em servidores externos.

---

## 2. Uso de inteligência artificial (IA)

Este aplicativo pode utilizar **modelos de inteligência artificial executados localmente (offline)** para análise de imagens ou objetos.

- Todo o processamento ocorre **exclusivamente no dispositivo do usuário**.
- O aplicativo **não envia imagens para servidores externos**.
- O aplicativo **não realiza reconhecimento facial para identificação de pessoas reais** com fins comerciais ou de vigilância.

---

## 3. Permissões do aplicativo

O aplicativo pode solicitar as seguintes permissões:

- **Câmera**: para capturar fotos conforme solicitado pelo usuário.
- **Armazenamento**: para salvar imagens no dispositivo do usuário.

Essas permissões são utilizadas **somente quando necessárias** e **exclusivamente para funcionalidades explícitas do aplicativo**.

---

## 4. Compartilhamento de dados

O **Ncg Camera** **não compartilha dados com terceiros**, incluindo:

- Anunciantes
- Plataformas de análise
- Serviços externos
- Redes sociais

---

## 5. Armazenamento e segurança

- Todas as imagens são armazenadas **localmente no dispositivo do usuário**.
- O aplicativo não mantém banco de dados remoto.
- O usuário pode apagar os dados do aplicativo a qualquer momento através das configurações do sistema.

---

## 6. Privacidade de crianças

Este aplicativo **não é direcionado a crianças menores de 13 anos** e **não coleta intencionalmente informações pessoais de crianças**.

Caso você seja pai, mãe ou responsável legal e acredite que uma criança utilizou o aplicativo indevidamente, entre em contato conosco.

---

Recomendamos que o usuário revise esta página ocasionalmente para se manter informado sobre eventuais mudanças.

A data da última atualização será sempre informada no início deste documento.
